Set a Greeting

    How to Use

Select what voicemail you want to set

  1. Select whether you want to set a voicemail for a personal or corporate machine
  2. Fill in the information for the machine you want to set a voicemail for
    • Provide the password for your phone system
    • Choose an audio file that you want to be used a voicemail
    • Additionally personal machines require the owner's extension number
  3. After clicking submit, a call will be made to your number. The call must be picked up if your system has no answering machine set up. Otherwise it can be left to ring until it is sent to the answering machine
  4. Once a person or the answering machine has answered the call, the recording must be allowed to play fully to completion

Your answering machine message has now been set!
